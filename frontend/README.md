# Music Streaming Application

A full-stack MERN (MongoDB, Express, React, Node.js) application for music streaming.

## Features
- User authentication
- Music upload and streaming
- Admin dashboard
- Responsive design

## Technologies
- Frontend: React, Vite
- Backend: Node.js, Express
- Database: MongoDB
- File Storage: Local file system
